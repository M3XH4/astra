Astra is a zodiac horoscope web application built with React and Tailwind CSS.

## Features
- Zodiac sign selection
- Birthdate zodiac detection
- Daily horoscope API
- LocalStorage favorite zodiac
- Responsive UI
- Copy horoscope feature

## Tech Stack
- React
- Tailwind CSS
- JavaScript
- API Ninjas Horoscope API
- LocalStorage
- Vercel

## Setup
npm install
npm run dev
